Detailed Description

The IMasterRepository base abstract class.

Class used as a base of interfaces for repositories on client and server sides. The class use polymorphism to registrate existing repositories. The repositories can be located at client or server sides. The name of repository is defined by class of object, which will be processed (e.g. ChannelsRepository handle object of ChannelInfo class).
Each database has it`s own full set of implemented repositories .
